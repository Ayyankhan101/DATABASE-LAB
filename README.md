# Database Lab - SEM-4

Welcome to the Database Lab repository for Semester 4. This repository contains all the SQL scripts and documentation related to the database laboratory exercises.

## Overview

This repository contains the lab assignments and SQL scripts for the Database Lab course in the 4th semester. Each lab exercise is organized in separate SQL files with corresponding documentation.

## Repository Structure

```
DATABASE-LAB/
├── README.md
├── lab01_db.sql
└── ...
```

## Current Lab Exercise

### Lab 01: Database Fundamentals
- File: `lab01_db.sql`
- Topic: Basic database operations and SQL commands

## Prerequisites

Before running the SQL scripts in this repository, ensure you have:
- A SQL database engine (such as MySQL, PostgreSQL, SQLite, etc.)
- Basic knowledge of SQL commands
- A text editor or database management tool

## Setup Instructions

1. Clone this repository to your local machine:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the repository directory:
   ```bash
   cd DATABASE-LAB
   ```

3. Connect to your database system and execute the SQL files:
   ```bash
   # Example for MySQL
   mysql -u username -p < lab01_db.sql
   ```

## How to Use

Each lab exercise is contained in a separate `.sql` file. To use these files:

1. Open the SQL file in a text editor to review the commands
2. Connect to your database system
3. Execute the SQL commands either by importing the file or copying the commands
